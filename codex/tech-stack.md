# Tech Stack

## Core Technologies

### Language & Runtime
- **Python 3.8+**: Primary programming language
- **Virtual Environment (venv)**: Dependency isolation

### AI & LLM
- **OpenAI GPT-4**: Large language model for natural language understanding
- **OpenAI Embeddings API**: Text embeddings for RAG

### Orchestration Framework
- **LangChain**: Agent orchestration and tool management
- **LangChain-OpenAI**: OpenAI integration for LangChain
- **LangChain-Community**: Additional tools and integrations

### Vector Store & RAG
- **ChromaDB**: Vector database for semantic search
- Persistent storage on disk
- Embedding-based retrieval

### Persistent Storage
- **Firebase Admin SDK**: Python SDK for Firebase
- **Cloud Firestore**: NoSQL database for engagement history
- Document-based storage model

### Utilities
- **python-dotenv**: Environment variable management
- **Pydantic**: Data validation and settings management
- **Requests**: HTTP library for API calls

## External Services

### Required
- OpenAI API (GPT-4 access required)
- Firebase/Firestore project with service account

### Optional
- Custom vulnerability scanning tools (extensible)

## Development Tools
- Git for version control
- pip for package management
- Virtual environment for dependency isolation
