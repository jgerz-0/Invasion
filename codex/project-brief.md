# Project Brief: PenTest AI Agent Backend

## Overview
A sophisticated Penetration Testing AI Agent backend built with Python, OpenAI's API, and LangChain. The system provides an intelligent assistant for security professionals conducting penetration tests.

## Core Purpose
Enable security professionals to conduct penetration tests more efficiently by leveraging AI for:
- Intelligent guidance on security testing approaches
- Knowledge retrieval of penetration testing techniques
- Vulnerability scanning and analysis
- Engagement history and context management

## Target Users
- Penetration testers
- Security professionals
- Red team operators
- Security consultants

## Business Value
- Accelerate penetration testing workflows
- Provide consistent testing methodology guidance
- Maintain comprehensive engagement documentation
- Enable knowledge sharing across security teams

## Success Criteria
- Agent successfully processes natural language commands
- RAG system retrieves relevant security knowledge accurately
- Engagement history persists across sessions
- Tool execution completes within acceptable timeframes (2-10 seconds)
- Easy to extend with custom security tools
